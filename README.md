# Bidsync

Bidsync es una plataforma diseñada para gestionar subastas y ofertas de manera eficiente. Este repositorio contiene dos partes principales:

- **Cliente**: Una aplicación web desarrollada con React, TypeScript y Vite.
- **Servidor**: Una API construida con el framework NestJS.

## Estructura del proyecto

```
Bidsync/
├── client/   # Aplicación cliente
└── server/   # API del servidor
```

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/R4F405/Bidsync.git
   ```

2. Sigue las instrucciones específicas en los directorios [`client`](https://github.com/R4F405/Bidsync/blob/main/client/README.md) y [`server`](https://github.com/R4F405/Bidsync/blob/main/server/README.md) para configurar cada parte.
